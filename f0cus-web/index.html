About F.O.C.U.S

F.O.C.U.S. is a malware prevention framework integrating machine learning to reduce risks, made with Python.

Our Mission

F.O.C.U.S. scans all ports, closing unauthorized ones. Clients specify whitelisted ports in the "whiteports" text file.
